could you give more details, I had a quick test it seems the norm works fine with the latest codebase:

>>> from monai.networks.layers.factories import Act, Norm
>>> from monai.networks.nets import UNet
>>> net = UNet(2, 1, 1, (4,8,16), (2, 2), norm=Norm.BATCH)
>>> net(torch.zeros(1, 1, 64, 64))
tensor([[[[-0.3094, -0.3094, -0.3094,  ..., -0.3094, -0.3094, -0.3094],
          [-0.3094, -0.3094, -0.3094,  ..., -0.3094, -0.3094, -0.3094],

I think maybe the batchnorm doesn't work, you can try other normalization with additional parameters such as

net = UNet(2, 1, 1, (4, 8, 16), (2, 2), norm=(Norm.GROUP, {'num_groups': 2, 'affine': False}))
